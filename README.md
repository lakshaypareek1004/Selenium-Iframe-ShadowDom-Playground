# Selenium-Iframe-ShadowDom-Playground
A fully interactive HTML playground for practicing Selenium automation with C#. Includes iframe with ID, iframe without ID, and iframe nested inside Shadow DOM. Perfect for mastering frame switching, ShadowRoot access, and stable locator strategies.
<img width="1918" height="1077" alt="image" src="https://github.com/user-attachments/assets/4027cdac-3700-4e4e-8208-862ea6805630" />

✔ Handling **iFrames with ID**  
✔ Detecting **iFrames without ID**  
✔ Switching into **iFrames inside Shadow DOM**  
✔ Building **stable, attribute-independent locators**  
✔ Writing **robust end-to-end automation tests**  

It visually demonstrates three different DOM embedding techniques commonly found in real production apps — making it an ideal resource for **QA Engineers**, **Selenium learners**, and **interview preparation**.
